Инструкция по запуску:
1. В папке SimbirSoft запустить хаб с помощью Selenium Grid Hub. Дождаться запуска хаба.
2. В папке SimbirSoft запустить узел(Node) на хабе с помощью Selenium Grid Node. Дождаться запуска узла.
3. В IntelIIJ Idea запустить выполнение кода с помощью команды "mvn clean test" (необходим установленный сборщик Maven) в секции Terminal.
4. Дождаться окончания хода тестирования.
5. С помощью секции Terminal запустить формирование отчетов Allure командой "mvn allure:serve".
6. После окончания работы с отчетом следовать инструкциям в терминале IntelIIJ Idea.
